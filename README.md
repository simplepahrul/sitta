# SISTEM INFORMASI TIRAS DAN TRANSAKSI BAHAN AJAR (SITTA)

Sistem informasi berbasis web untuk simulasi pengelolaan tiras dan transaksi bahan ajar.

---

## 🚀 Live Demo

🔗 https://simplepahrul.github.io/sitta  

---

## 📌 Deskripsi Sistem

SITTA (Sistem Informasi Tiras dan Transaksi Bahan Ajar) adalah sistem berbasis web yang digunakan untuk mengelola data tiras (stok bahan ajar) dan transaksi distribusi bahan ajar di lingkungan institusi pendidikan.

Sistem ini dirancang untuk meningkatkan efisiensi, akurasi, serta transparansi dalam pengelolaan data yang sebelumnya dilakukan secara manual.

---

## 🎯 Tujuan Pengembangan

Adapun tujuan dari pengembangan sistem ini adalah:

- Meningkatkan efisiensi pengelolaan data bahan ajar  
- Mempermudah pencatatan transaksi keluar dan masuk bahan ajar  
- Mengurangi kesalahan dalam proses administrasi manual  
- Mendukung digitalisasi sistem administrasi pendidikan  

---

## ⚙️ Fitur Utama

- Manajemen data stok bahan ajar  
- Pencatatan transaksi bahan ajar  
- Monitoring data tiras  
- Tampilan data yang sederhana dan mudah digunakan  

---

## 🧱 Teknologi yang Digunakan

Sistem ini dikembangkan menggunakan teknologi web dasar:

- HTML → Struktur halaman web  
- CSS → Tampilan dan desain antarmuka  
- JavaScript → Interaksi dan logika pada sisi pengguna  

Sistem dapat dijalankan langsung melalui browser tanpa instalasi tambahan.

---

## 📚 Manfaat Sistem

- Membantu digitalisasi administrasi bahan ajar  
- Mempercepat proses pencatatan dan pelaporan  
- Mengurangi kesalahan pencatatan manual  
- Menjadi dasar pengembangan sistem informasi lebih lanjut  

---

## 🧠 Catatan Pengembangan

Proyek ini dikembangkan sebagai bagian dari tugas akademik Universitas Terbuka, sekaligus sebagai implementasi pembelajaran sistem informasi berbasis web.

Pengembangan dilakukan dengan pendekatan bertahap untuk memastikan sistem tetap sederhana, stabil, dan mudah dipahami.

---
